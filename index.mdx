OpenLens is a community-maintained binary build repository that compiles and distributes signed, open-source builds of the [Lens Kubernetes IDE](https://github.com/lensapp/lens). Unlike the official Lens IDE distribution, OpenLens requires no login and includes only the open-source components licensed under MIT.

## Get started in minutes

<Steps>
  <Step title="Download OpenLens">
    Go to the [Releases page](https://github.com/MuhammedKalkan/OpenLens/releases) or use a package manager like Homebrew, Scoop, Winget, or Chocolatey to install OpenLens on your system.
  </Step>
  <Step title="Connect your cluster">
    Open OpenLens and add your kubeconfig. OpenLens reads `~/.kube/config` by default, or you can import a custom kubeconfig file through the UI.
  </Step>
  <Step title="Install extensions (optional)">
    Starting with v6.3.0, some features like the node/pod menu are available as separate extensions. Install `@alebcay/openlens-node-pod-menu` from the Extensions page.
  </Step>
</Steps>

<Note>
  Lens has closed its source code. No further OpenLens updates are expected. Use the existing releases for your Kubernetes management needs.
</Note>
